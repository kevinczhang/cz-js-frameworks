# Vue

 Unlike other monolithic frameworks, Vue is designed from the ground up to be incrementally adoptable.

## Quick start

```bash
npm install -g @vue/cli
# OR
yarn global add @vue/cli

vue create hello-world
```

##  Or use GUI

```text
vue ui
```
